Mulțime

Mulțimile (set) în limbajul C++ reprezintă un conteiner de elemente omogene unice(deja sortate). Set-ul face parte din biblioteca standartă ale limbajului C++ STL (Standard Template Library), deaceea pentru a utiliza set-ul este nevoie de adăugat biblioteca prin #include <set>.

Sintaxa declarației unei mulțimi este:

set <tip_element> numele_variabilei;

Exemplu:
  • set <int> numere;
  • Este declarată o variabilă cu denumire numere de tip mulțime, inițial reprezintând mulțime vidă.
Se poate la definirea mulțimii de inițializat mulțimea cu valori.
Exemplu:
  • set <char> vocale = {'a', 'o', 'u', 'i', 'e'};
  • Este declarată o variabilă cu denumire vocale de tip mulțime cu valorile de vocale de tip char.
Pentru operarea eficientă cu mulțimile este nevoie de cunoscut o lista de funcții.
Pentru exemplificare propunem o problemă: Într-un șir de caractere de numărat numărul vocalelor.
  • #include <iostream>
  • #include <set>
  • #include <string>
  • using namespace std;
  • int main()
  • {
  • set <char> vocale = {'a', 'o', 'u', 'i', 'e'};
  • string sir;
  • cout<<"Introduceti sir de caractere:"<<endl;
  • getline(cin,sir);
  • int nr_vocale=0;
  • for (int i = 0; i < sir.size(); i++) {
  • if (vocale.find(sir[i]) != vocale.end()) nr_vocale++;
  • }
  • cout<<"Numarul de vocale = "<<nr_vocale<<endl;
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Introduceti sir de caractere:

cautam vocalele

Numarul de vocale = 7
