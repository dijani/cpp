Introducere

Prima cunoștința cu programul în C++ trebuie făcută pe baza de un program simplu de salutare.

  1. #include <iostream>
  2. using namespace std;
  3. int main()
  4. {
  5. cout<<"Salutare!";
  6. return 0;
  7. }
Rezultatul execuției acestui program va fi:

Salutare!

Fiecare rând din programul dat realizează următoarele:

  1. Se conectează fișierul iostream cu biblioteca de subprograme de intrare/ieșire cu ajutorul directivei include.
  2. Se indică utilizarea unui spațiu de nume std pentru identificatorii ce vor fi utiizate în program.
  3. Se declară funcția de bază cu numele main și cu tipul rezultatului int în care se vor include instrucțiunile necesare rezolvării problemei propuse.
  4. Începutul instrucțiunii compuse ce permite realizarea unui bloc de instrucțiuni.
  5. Se va afișa pe ecran mesajul: Salutare! ce este inclus între ghilimele.
  6. Se returnează valoarea 0 în urma execuției funcției, ceea ce va indica că funcția s-a executat fără erori.
  7. Sfârșitul instrucțiunii compuse/unui bloc de instrucțiuni.
