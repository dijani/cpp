Tipuri simple

În limbajul C++ sunt următoarele tipuri simple de date: int, float, char, bool, enum, void. Fiecare tip de date descrie data cu ajutorul a 2 caracteristici: mulțimea de valori ale datei și mulțimea de operații ce pot fi efectuate asupra datei. Pentru a studia fiecare dintre tipurile propuse alegeți butonul corespunzător.



Tipul de date int definește mărimile ce pot lua valorile întregi (pozitive și negative) din intervalul de la −2.147.483.648 până la 2.147.483.647, ocupând 4 octeți în memoria calculatorului. Intervalul acesta reprezintă mulțimea de valori ale tipului dat.
Mulțimea de operații pentru tipul dat este formată din următoarele grupe de operații:
  • aritmetice: + (adunare), - (scăcere), * (înmulțire), / (câtul împărțirii), % (restul împărțirii)
  • relaționale: < (mai mic),> (mai mare), == (egal), <= (mai mic sau egal),>= (mai mare sau egal), != (neegal)
Rețineți!
Dacă asupra datelor de tip int vom aplica operațiile aritmetice rezultatul obținut va fi la fel de tipul int. Însă dacă vom aplica operațiile relaționale rezultatul deja va fi de tipul bool.
Pentru exemplificare propunem o problemă: Se dă un număr din 2 cifre, de aflat suma cifrelor acestui număr.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • int n,s;
  • cout<<"Dati un numar din 2 cifre"<<endl;
  • cin>>n;
  • s=n/10+n%10;
  • cout<<"Suma cifrelor="<<s;
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Dati un numar din 2 cifre

23

Suma cifrelor=5

Sarcina:
Încercați să realizați de sine statator un program ce va rezolva următoarea problemă:
Se dau 2 numere întregi ce reprezintă laturile unui dreptunghi. Calculați aria dreptunghiului.

Tipul de date float definește mărimile ce pot lua valorile zecimale (pozitive și negative) din intervalul de la −3.4E38 până la 3.4E38, ocupând 4 octeți în memoria calculatorului. Intervalul acesta reprezintă mulțimea de valori ale tipului dat. Numerele zecimale folosesc punct zecemal pentru separarea părții întregi de la partea fracțională. La fel se utilizează factor scală (e/E) pentru a interpreta *10 în scrierea numerelor în format științific.
Mulțimea de operații pentru tipul dat este formată din următoarele grupe de operații:
  • aritmetice: + (adunare), - (scăcere), * (înmulțire), / (împărțire)
  • relaționale: < (mai mic),> (mai mare), == (egal), <= (mai mic sau egal),>= (mai mare sau egal), != (neegal)
Rețineți!
Dacă asupra datelor de tip float vom aplica operațiile aritmetice rezultatul obținut va fi la fel de tipul float. Atenția deosebită trebuie acordată operației de împărțire. Dacă ambii operatori sunt de tipul int atunci rezultatul operației la fel va fi valoarea de tip int. Pentru obținerea împărțirii reale asupra 2 operatorii întregi trebuie de forțat aplicarea operației reale sau de modificat un operator din int în float (indicând cuvântul float sau prin *1.0).
La aplicarea operațiilor relaționale rezultatul va fi de tipul bool.
Pentru exemplificare propunem o problemă: Se dau 2 numere întregi, de aflat media aritmetică a acestora.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • int n1,n2;
  • cout<<"Dati 2 numere"<<endl;
  • cin>>n1>>n2;
  • float m;
  • m=float(n1+n2)/2;
  • cout<<"Media numerelor="<<m;
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Dati 2 numere

3

4

Media numerelor=3.5

Sarcina:
Încercați să realizați de sine statator un program ce va rezolva următoarea problemă:
Se dau 3 numere ce reprezintă bazele unui trapez și înățlimea lui. Calculați aria trapezului.

Tipul de date char definește mărimile ce pot lua valorile caracteriale din tabelul ASCII, ocupând 1 octet în memoria calculatorului. Tabelul ASCII conține 256 de valori numerotate de la 0..255 și reprezintă mulțimea de valori ale tipului dat. Fiecare caracter este delimitat de apostrofuri(exemplu 'A'), fapt ce permite deosebirea valorilor acestui tip de la valorile altor tipuri. Operarea cu valorile acestui tip poate fi făcută direct prin indicarea poziției din tabelul ASCII.
Mulțimea de operații pentru tipul dat este formată din următoarele:
  • relaționale: < (mai mic),> (mai mare), == (egal), <= (mai mic sau egal),>= (mai mare sau egal), != (neegal)
Rețineți!
Dacă operăm cu valorile de tip char la nivel de poziție al tabelului ASCII, dar pozițiile reprezintă numere întregi, atunci este permis să aplicăm operațiile aritmetice. Dacă după efectuarea operațiilor aritmetice dorim să accesăm valorile de tip caracterial este nevoie de utilizat cuvântul char în calitate de convertor de valori.
La aplicarea operațiilor relaționale rezultatul va fi de tipul bool.
Pentru exemplificare propunem o problemă: Se dă un caracter. Da aflat caracterul ce se află peste 3 poziții spre dreapta, cât și poziția lui în tabelul ASCII.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • char c1,c2;
  • cout<<"Dati caracterul"<<endl;
  • cin>>c1;
  • c2=char(c1+4);
  • cout<<"Caracterul gasit="<<c2<<endl;
  • int p;
  • p=int(c2);
  • cout<<"Pozitia caracterului="<<p;
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Dati caracterul

A

Caracterul gasit=E

Pozitia caracterului=69

Sarcina:
Încercați să realizați de sine statator un program ce va rezolva următoarea problemă:
Se dă un număr întreg. De afișat caracterul ce se află pe poziția dată în tabelul ASCII.

Tipul de date bool definește mărimile logice cu valorile: fals (0/false) și adevărat (1/true), ocupând 1 octet în memoria calculatorului. Acestea 2 valori și reprezintă mulțimea de valori ale tipului dat.
Mulțimea de operații pentru tipul dat este formată din următoarele grupe de operații:
  • logice: ! (not/nu), && (and/și), || (or/sau).
  • relaționale: < (mai mic),> (mai mare), == (egal), <= (mai mic sau egal),>= (mai mare sau egal), != (neegal)
Rețineți!
Aplicând toate operațiile ale tipului de date bool obținem rezultate de tip bool. Mai mult ca atât aplicând operațiile relaționale asupra valorilor de celelalte tipuri rezultatul tot va fi de tipul bool. Deaceea datele de tip logic sunt utilizate în mare măsură pentru crearea expresiilor logice sau pur și simplu condițiilor.
Pentru exemplificare propunem o problemă: Se dau 2 numere. Da aflat valoarea de adevar a următoarei afirmației: primul număr este mai mare decât al doilea.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • int n1,n2;
  • cout<<"Dati 2 numere"<<endl;
  • cin>>n1>>n2;
  • bool b;
  • b=n1>n2;
  • cout<<"n1>n2? "<<b<<endl;
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Dati 2 numere

5

6

n1>n2? 0

Sarcina:
Încercați să realizați de sine statator un program ce va rezolva următoarea problemă:
Se dă un caracter. De afișat valoarea de adevar a următoarei afirmației: Caracterul introdus este o literă mică alfabetului latin.

Verifică cunoștințele:

Tipul de date enum este un tip definit de utilizator care descrie un șir de valori (la dorința utilizatorului) ce capătă numerele de ordine începând cu 0. Mulțimea valorilor din șirul descris și reprezintă mulțimea de valori ale tipului dat. Există posibilitatea de a atașa numerele de ordine la dorința utilizatorului, însă lucrul acesta trebuie definit de la bun start în lista inițială.
  • Sintaxa declarației tipului:
  • enum numetip={id1, id2, ..., idn};
  • Sintaxa declarației variabilei:
  • numetip numevar;
  • Sintaxa declarației comună (tip și var)
  • enum numetip={id1, id2, ..., idn} numevar;
  • Sintaxa declarației tipului cu stabilirea numerelor de ordine proprii:
  • enum numetip={id1, id2=5, ..., idn};
  • Numerele de ordine ce nu sunt indicate de către utilizator se calculează incrementând cea precedentă
Mulțimea de operații pentru tipul dat este formată din următoarele:
  • relaționale: < (mai mic),> (mai mare), == (egal), <= (mai mic sau egal),>= (mai mare sau egal), != (neegal)
Rețineți!
În expresii aritmetice datele de tip enum sunt tratate ca numere întregi (numerele lor de ordine din listă). Conversia către int este implicită, dar conversia inversă către enum trebuie realizată de către utilizator.
Tipul enum este utilizat pentru a da mai multă claritatea programului, reprezentând niște codificări numerice.
Tipul de date void este un tip specific ce nu are valori și nu prevede operații asupra datelor. Astfel mulțimea de valori și mulțimea de operații sunt mulțimile vide. Variabilele de tip void nu se declară.
Rețineți!
Tipul void se utilizează în special pentru funcții și pointeri.
