Date

Orice informația supusă prelucrării se transformă în dată. Datele sunt reprezentate sub forma de mărimile: constantă și variabilă. Constantă este o mărime ce își păstrează valoarea sa pe tot parcursul execuției programului. Variabilă este o mărime ce poate schimba valoarea sa pe parcusul execuției programului. Pentru operarea eficientă în programul orice mărime fie constantă, fie variabilă este caracterizată de identificatorul (denumirea), valoarea și tipul.
Exemplu:
  • const float pi=3.14;
  • Este o constană cunoscută π
  • "pi" reprezintă identificatorul sau denumirea mărimii
  • "3.14" reprezintă valoarea mărimii
  • Valoarea fiind 3.14 implică caracterizarea mărimii prin tipul de date numeric zecimal (float)
  • int aria=7;
  • Este o variabilă
  • "aria" reprezintă identificatorul sau denumirea mărimii
  • "7" reprezintă valoarea mărimii
  • Valoarea fiind 7 implică caracterizarea mărimii prin tipul de date numeric întreg (int)
Datele pot să fie simple și structurate. Data simplă operează cu o singură valoare, pe când data structurată operează cu o mulțime de valori. Deaceea deosebim și tipurile de date simple și structurate. Altă proprietate a datei este timpul utilizării memoriei pentru operarea cu valoarea. Deosebim datele statice și dinamice. Pentru data statică zona de memorie este rezervată pentru tot timpul cât programul se execută, astfel aceasta zonă nu poate fi utilizată de alte date, chiar și dacă nu este utilizată la moment de către program. Pentru data dinamică zona de memorie nu se rezervează imediat, ci în momentul unei comenzi speciale de rezervare a memoriei; memorie este ocupată de valoarea datei atâta timp cât este necesar de program, cum numai programul nu are nevoie de valoarea dată se utilzează comanda de distrugerea mărimii în urma cărei zona de memorie se eliberează.
