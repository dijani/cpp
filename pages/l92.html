Șiruri de caractere

Șir de caractere reprezintă o colecție de date caracteriale. Există 2 modalități de manipulare cu mulțimea de caractere. Primul reprezintă un tabel liniar din elemente de tip char(ceea ce presupune operarea cu o lungime fixa - structura statică) și după ultimul caracter se află un caracter special '\0' - numit caracter null. Lucrul cu elementele acestei modalități de structură este mai rapid, însă nu oferă multe funcții încorporate pentru a manipula cu șiruri.

Sintaxa șirului prin tabel:

char numele_variabilei [lungimea];
Exemplu:
  • char sir[5];
  • Este declarat un șir din 5 caractere cu denumire sir, indexarea fiind de la 0 până la 4, poziția a 5-a este ocupată de caracterul \0
A doua modalitate represintă o colecție de structură string, ce nu presupune rezervarea dimensiunii la declararea structurii date, astfel deja reprezintă o structură dinamică. Utilizarea acestei modalități implică indicarea unei biblioteci speciale în program #include<string>.
Clasa String operează cu obiecte care reprezintă fluxuri de caractere. Operarea cu obiecte este mai lentă decăt cu tabele, însă există o serie de funcționalități asupra șirurilor de caractere, ceea ce oferă lucru mai eficient în operarea cu șiruri.

Sintaxa declarației string-ului:

string numele_variabilei;
Exemplu:
  • string salutare;
  • Este declarat un șir de caractere cu denumire salutare
La citirea șirurilor de caractere trebuie să fim atenți la prezența " " (spațiului) în cadrul șirului.
La aplicarea comenzii cin>>sir; se va citi numai primul cuvânt din șir, adică până la primul spațiu întâlnit. Dacă apare necesitate de citirea unui șir de caractere cu spații se utilizează alta comanda pentru citire: getline (cin, sir); se utilizează mai des pentri string-uri și cin.getline(sir,5) se utilizează pentru tabele de char, specificând ca al doilea argument lungimea tabelului.

Asupra șirurilor de caractere se admit operațiile de comparație, comparația fiind realizată caracter cu caracter în conformitate cu codurile numerice ale caracterelor din tabelul ASCII.
Exemplu:
  • string salutare1="salut", salutare2="hello";
  • cout<<salutare1<salutare2;
  • Sunt declarate 2 șiruri cu valorile salut și hello
  • Compararea acestor 2 șiruri va afișa rezultatul 0, deoarece primul caracter din primul șir "s" nu este mai mic decât primul caracter din șirul al dolilea "h". Numărul de ordine al caracterului "s" este mai mare decât numărul de ordine al caracterului "h".
Pentru exemplificare propunem o problemă: Fie este dat un șir de caractere și o literă. Verificați de câte ori litera dată apare în șirul propus.
  • #include <iostream>
  • #include <string>
  • using namespace std;
  • int main()
  • {
  • string sir;
  • cout<<"Introduceti sirul:"<<endl;
  • getline(cin,sir);
  • char caracter;
  • cout<<"Introduceti caracterul:"<<endl;
  • cin>>caracter;
  • int apare=0;
  • for (int i = 0; i < sir.size(); i++)
  • if (sir[i] == caracter) apare++;
  • cout<<"Caracterul "<<caracter<<" apare de "<<apare<<" ori";
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Introduceti sirul:

interfata

Introduceti caracterul:

a

Caracterul a apare de 2 ori
