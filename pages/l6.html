Instrucțiuni de ramificare

Instrucțiunile de ramificare în limbaje de programare servesc pentru dirijarea fluxului execuției programului. Algoritmii nu întotdeauna sunt de tip liniar. Există situații când pații necesari rezolvarii unei problemei să presupună selectarea unor acțiuni din lista acțiunilor posibile în funcție de diferite situații înconjurătoare, ceea ce impune execuția condiționată/selectivă sau pur și simplu ramificare.
În limbajul C++ există 2 instrucțiuni ce permit realizarea astfel de situații: If și Switch.

If

Instrucțiunea If reprezintă o ramificare simplă. Operatorul dat verifică o condiție (expresie logică/booleană) și în funcție de justețea/falsitatea propune selectarea a 2 execuții diferite după necesitatea problemei.

Sintaxa:

if (condiție){
//blocul acțiunilor ce se execută în cazul condiției adevărate
}
else {
//blocul acțiunilor ce se execută în cazul condiției false
}
Atenție
Ramura else este opțională și deaceea poate lipsi.
Exemplu:
  • if (a>0) {a+=5;} else {a/=2;}
  • Se evaluează condiția a>0 ceea ce corespunde afirmației că numărul a este pozitiv.
  • Dacă valoarea numărului a satisface condiția fiind număr pozitiv se va efectua acțiunea din prima acoladă, adică valoare lui a se va mări cu 5.
  • Dacă valoarea numărului a nu satisface condiția fiind un număr nepozitiv (nul sau negativ) se va efectua acțiunea din acolada după cuvântul else, adică valoarea lui a se va micșora de 2 ori.
  • if (a%2==0) {cout<<"Numărul este par";}
  • Se evaluează condiția a%2==0 ceea ce corespunde afirmației că restul împărțirii numărului a la 2 este 0, adică numărul a se împarte la 2 fără rest.
  • În cazul cînd valoarea variabilei a satisface această condiție de va realiza comanda din acolade, adică pe ecran se va afișa mesajul dintre apostrofuri.
  • În lipsa ramurii else în cazul nesatisfacerii condiției date nu se va realiza nici o acțiune.

Forma scurtă operatorului If (operatorul ternar)

Sintaxa:
variabila=(condiție) ? (acțiunea_True) : (acțiunea_False);
corespunde sintaxei if complete:
if (condiție) {acțiunea_True} else {acțiunea_False}
Exemplu:
  • max = (n1 > n2) ? n1 : n2;
  • Se evaluează condiția n1>n2, adică se compară valorile cu scopul aflarii celui mai mare.
  • Dacă condiția este satisfăcută variabila max va obține ca rezultat valoarea n1 (prima după semnul ?)
  • Dacă condiția nu este satisfăcută variabila max va obține ca rezultat valoarea n2 (valoarea după semnul :)

Exerciții:

Formulați condiții:
  • a este mai mare decât 5
  • b este diferit de 10
  • c este mai mic decât 6 și mai mare decât 2
  • diferența dintre a și b este un număr impar
Pentru variabilele propuse x=9, y=4, a=2 stabiliți valoarea lui a după execuția instrucțiunii.
  • if (x>0) a=0;
  • if (!y<=0) a=5 else a=2;
  • a=(x>y)? 10 : 20;
Pentru exemplificare propunem o problemă: Se dau numere reale a și b. Să se afișeze suma acestora dacă numerele sunt egale și diferența lor în caz contrar.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • float a,b;
  • cout<<"Dati 2 numere"<<endl;
  • cin>>a>>b;
  • if (a==b) cout<<"Suma numerelor="<<(a+b);
  • else cout<<"Diferenta numerelor="<<(a-b);
  • return 0;
  • }
Rezultatul execuției acestui program va fi:
Cazul 1:

Dati 2 numere

5

5

Suma numerelor=10

Cazul 2:

Dati 2 numere

3

1

Diferenta numerelor=2

Switch

Instrucțiunea Switch reprezintă o ramificare compusă. Operatorul dat lucrează cu un selector valoarea căruia se compară la egalitate cu o listă de cazuri. Dacă se obține o egalitate adevărată se efectuează acțiunea indicată pentru cazul dat.

Sintaxa:

switch (expresie){
case v1: {acțiune; break;}
case v2: {acțiune; break;}
case v3: {acțiune; break;}
...
default: {acțiune;}
}
Atenție

Cazul default joacă rol ramurii else in cazul operatorului If. Dacă nu se găsește nici o egalitate a valorii selectorului cu unul dintre cazurile propuse se fixează cazul neegalității sau caz alternativ ce interpretează imposibilitatea realizării selecției.

Structura Switch permite gruparea unor cazuri pentru a fi realizată o acțiune comună. În așa situație se indică un șir de cazuri cu constante și semnele ”:”. Acțiunea ce trebuie realizată se scrie o singura dată după ultimul caz scris.

Sintaxa:

switch (expresie){
case v1:
case v2:
...
case vk: {acțiune; break;}
...
default: {acțiune;}
}

Pentru exemplificare propunem o problemă: Se dau numere reale a și b și un caracter o ce indică o operație aritmetică. Să se afișeze rezultatul operației indicate.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • float a,b;
  • char o;
  • cout<<"Dati 2 numere"<<endl;
  • cin>>a>>b;
  • cout<<"Indicati operatia (+,-,*,/)"<<endl;
  • cin>>o;
  • switch (o){
  • case '+' : cout<<"a+b="<<(a+b); break;
  • case '-' : cout<<"a-b="<<(a-b); break;
  • case '*' : cout<<"a*b="<<(a*b); break;
  • case '/' : cout<<"a/b="<<(a/b); break;
  • default: cout<<"Operatia nerecunoscuta!";}
  • return 0;
  • }
Rezultatul execuției acestui program va fi:

Dati 2 numere

4.5

6.2

Indicati operatia (+,-,*,/)

+

a+b=10.7
