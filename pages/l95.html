Fișiere

Deja cunoaștem că datele inițiale în programele noastre ajung prin așa numite flux-uri de intrare stream-uri, iar datele finale din programele noastre se extrag pe ecran la fel printr-un flux de ieșire. Anume pentru dirijarea cu fluxurile date noi ne folosim de operatorii cin - consola input (tastatura) și cout - consola output (ecran). Ca să putem utiliza acești operatori obligatoriu se indică biblioteca iostream (Input Output Stream) ce răspunde de fluxurile date.
Pe lângă fluxurile standarte se pot fi folosite și fluxurile de fișiere. În acest scop se utilizează biblioteca fstream (File Stream). Acest lucru înseamnă că datele (colecții de date) sunt stocate în exteriorul mediilor de programare. După necesitate noi putem să ne adresăm la acestea date sau pentru a aduce datele în program, sau pentru a extrage datele din program.

Fișiere reprezintă o structură de date prin intermediul căreia este organizată informația pe harddisk. Un fișier este o colecție de linii de text terminate printr-un caracter special EOL (End of Line - sfârșit de linie). Sfârșitul fișierului este specificat prin alt caracter special denumit EOF (End of File - sfârșit de fișier).
  • În lucrul cu fișiere sunt necesare 3 operații:
  • - deschiderea fișierului (se declară o variabilă pentru un fișier extern)
  • - citirea/scrierea datelor din/în fișier
  • - închiderea fișierului

Sintaxele operațiilor cu fișiere sunt:

  • #include<fstream>
  • conectarea bibliotecii pentru lucrul cu fișiere

  • ifstream nume_fișier_in("denumirea_fișierului_1");
  • se deschide fișierul extern cu denumirea_fișierului_1 pentru operația de citire prin intermediul variabilei nume_fișier_in

  • ofstream nume_fișier_out("denumirea_fișierului_2");
  • se deschide fișierul extern cu denumirea_fișierului_2 pentru operația de înscriere prin intermediul variabilei nume_fișier_out

  • nume_fișier_in>>variabila;
  • se citește valoarea din fișierul prin intermediul nume_fișier_in și se înscrie în variabilă

  • nume_fișier_out<<variabila;
  • se înscrie valoarea din variabilă în fișierul prin intermediul nume_fișier_out

  • nume_fișier.close();
  • se închide fișierul - nume_fișier
Exemplu:
  • ifstream in("date.in");
  • in>>a;
  • in.close();

  • se deschide pentru citire fișierul extern cu denumirea date.in prin intermediul variabilei in
  • din fișierul in valoarea se citește și se înscrie în variabila a
  • fișierul in se închide
Pentru exemplificare propunem o problemă: În fișierul date.in sunt stocate 2 numere întregi (10 15). Se cere de calculat suma acestor numere și de înscris în fișierul date.out.
  • #include <iostream>
  • #include <fstream>
  • using namespace std;
  • int main()
  • {
  • ifstream in("date.in");
  • int a,b;
  • in>>a>>b;
  • in.close();
  • ofstream out("date.out");
  • out<<(a+b);
  • out.close();
  • return 0;
  • }
Rezultatul execuției acestui program va fi stocat în fișiere:

date.in : 10 15

date.out : 25


Remarcă
La deschiderea fișierelor se indică numai denumirea fișierelor externe ce se află în același director cu fișierul-sursă *.cpp. În cazul când fișierele cu date se află în alte directorii se indică calea completă către fișierul cu date.

Sarcina:
Elaborați un program ce va realiza: Din fișierul note.txt se citesc cele 4 note obținute de un elev la informatică pe parcursul unui semestru și nota de la teza. Scrieți un algoritm care calculează media lui și să o scrie într-un fișier numit media.txt.
Pentru realizarea sarcinii puteți utiliza zona de mai jos a editorului, însă problema nu va funcționa complet fară fișiere externe. Sarcina dată este recomnadată să fie realizată la un compilator offline cu crearea fișierelor indicate în problemă.
