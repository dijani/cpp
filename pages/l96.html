Funcții

Orice problema poate fi divizată în subprobleme mici. Astfel orice program poate fi compus din mai multe subprograme. Subprogram reprezintă un fragment al programului (program principal). Utilizarea subprogramelor face textul programului mai scurt și mai ușor de conceput. Un avantaj al subprogramelor constă în faptul că o dată definit el poate fi utilizat de număr dorit de ori (apel de subprogram). În limbajul C++ subprogramele sunt reprezentate sub forma de funcții.

Cu o funcție main noi deja suntem cunoscuți. Ea reprezintă blocul principal al programului în C++. În afară de aceasta noi putem defini funcții proprii. Orice funcție trebuie definită înainte de blocul funcției main (există posibilitatea de definire și mai jos de blocul main), cu o structură asemănătoarea cu main: antetul funcției(prima linie) și corpul ei.

Sintaxa declarației de funcție este:

  • tip_rezultat nume_functie(parametrii_formali) {
  • corpul funcției
  • }
Exemplu:
  • int suma (int a, int b) {
  • return a+b;
  • }

  • se definește o funcție cu numele suma cu 2 parametri formali întregi a și b (argumentele funcției), rezultatul funcției este de tip întreg
  • în corpul funcției se realizează operația de adunare și rezultatul se întoarce în main prin operatorul return
După ce funcția este definită, în blocul main ea trebuie chemată pentru execuția (apelarea funcției). Funcția se apelează utilizând numele ei și parametrii actuali (valorile concrete ce se transmit la funcția).
Exemplu:
  • int suma (int a, int b) {
  • return a+b;
  • }
  • int main() {
  • int x,y;
  • cout<<"Introdu 2 valori intregi:"<<endl;
  • cin>>x>>y;
  • cout<<"Suma acestor numere este:"<<suma(x,y);
  • return 0;
  • }

  • după definirea funcției înainte de blocul main, ea este chemată în operatorul cout
  • apelul este compus din numele funcției suma și 2 parametri actuali x și y
Remarcă
Funcțiile des sunt utilizate pentru realizarea calculelor și obținerea unui rezultat ce trebuie întors în blocul main.
Însă este posibil să definim o funcție ce nu realizează calcule și nu întoarce nici un rezultat în blocul main. Astfel de funcții la tipul rezultatului au tipul void.
Exemplu:
  • void salutare (string s) {
  • cout<<"Salut, " + s;
  • }

  • se definește o funcție cu numele salutare cu 1 parametru formal de tip șir de caractre s (argumentul funcției), care nu returnează nimic (void)
  • în corpul funcției se utilizează operatorul cout ce afișează rezultatul concatenării a 2 șiruri: Salut, și valoarea argumentului s
Apelul astfel de funcții se deferă de apelul funcției cu rezultat.
Exemplu:
  • void salutare (string s) {
  • cout<<"Salut, " + s;
  • }
  • int main() {
  • salutare("Maria");
  • return 0;
  • }

  • după definirea funcției înainte de blocul main, funcția salutare este chemată cu parametrul actual Maria ca instrucțiunea independentă

În continuare este prezentat un exemplu de program cu câteva funcții.
  • #include <iostream>
  • using namespace std;
  • bool min(int a, int b){
  • return a < b;
  • }
  • void salutare() {
  • cout<<"Salut, prietene!"<<endl;
  • }
  • int main()
  • {
  • salutare();
  • int x,y;
  • cout<<"Introdu 2 numere întregi=>"<<endl;
  • cin>>x>>y;
  • bool b = min(x,y);
  • cout<<"Primul numar e mai mic decat al doilea? "<<b;
  • return 0;
  • }
Rezultatul execuției acestui program va fi următorul:

Salut, prietene!

Introdu 2 numere întregi

3 4

Primul numar e mai mic decat al doilea? 1
