Structuri

Structurile reprezintă o colecție finită de valori neomogene. Fiecare element al structurii este un câmp ce trebuie caracterizat prin nume și tip.

Sintaxa declarației unei structuri este:

struct nume_tip {
tip1 nume_câmp1;
tip2 nume_câmp2;
...
} nume_var;

Exemplu:
  • struct elev {
  • string nume, prenume;
  • float note[10];
  • } e1,e2;

  • S-a definit tipul structură cu denumire elev, ce este alcătuit din 3 câmpuri:
  • 2 câmpuri de tip șir de caractere cu denumirile nume și prenume
  • 1 câmp de tip tabel liniar cu denumire note din 10 numere reale
  • după definirea structurii s-a declarat 2 variabile cu denumirile e1 și e2
Există mai multe variații de declarație a structurilor. În sintaxa de mai sus a fost prezentată cea mai răspândită varianta. Însă structura poate să nu conțină denumirile de variabile la sfârșit. Atunci declararea variabilelor se face în mod obișnuit indicând numele tipului și denumirea variabilei. De exemplu aceeași structura elev poate fi utilizată pentru declararea variabilelor mai pe urmă.
Exemplu:
  • struct elev {
  • string nume, prenume;
  • float note[10];
  • };
  • elev e1, e2;
La definirea structurii poate lipsi denumirea tipului, aceasta modalitatea impune indicarea variabilelor în mod obligatoriu și imposibilitatea declarării variabilelor în plus în textul programului mai jos.
Exemplu:
  • struct{
  • string nume, prenume;
  • float note[10];
  • } elev;

  • elev în cazul dat este variabila de tip structură anonimă
Să ne jucăm pentru a fixa cunoștințele
Referirea la câmpurile structurii se face indicând numele variabilei și numele câmpului separate prin "." .
Exemplu:
  • elev.nume
  • s-a accesat valoarea câmpului nume ale variabilei elev de tip structură

  • elev.note[0]
  • s-a accesat valoarea câmpului note și anume a primului element al tabelului ale variabilei elev de tip structură
Asupra structurilor se permite numai operația de atribuire. Aceasta se aplică numai pentru structurile de aceeași componență și realizează copierea conținutului în întregime dintr-o structură în altă.
Exemplu:
  • În prezența declarației:
  • struct carte {
  • string titlu;
  • float pret;
  • } c1,c2;

  • c2 = c1;
  • se copie conținutul dîn structura c1 în structura c2
Pentru exemplificare propunem o problemă: Se cunoaște o listă de 10 manuale descrise prin titlu și preț. Se cere să aflăm câte manuale sunt mai scumpe decât 35 de lei.
  • #include <iostream>
  • using namespace std;
  • int main()
  • {
  • struct carte{
  • string titlu;
  • float pret;
  • } manuale[10];
  • int n = 0;
  • for (int i=0; i < 10; i++){
  • cin>>manuale[i].titlu;
  • cin>>manuale[i].pret;
  • manuale[i].pret > 35 ? n++ : n ;
  • }
  • cout<<"Numărul manualelor mai scumpe decât 35 lei = "<<n;
  • return 0;
  • }
